# MASH Diffusion

## Install Blender

```bash
https://www.blender.org/download/
->
# MacOS
'/Applications/Blender.app/Contents/MacOS/Blender'
# Linux
$HOME/Install/blender/blender
```

## Setup

```bash
conda create -n md python=3.10
conda activate md
./setup.sh
```

## Run

```bash
python sample_cfm.py
```
